Blog-termux

纯静态单页面应用 — 无需 PHP / Node / Python 后端

Nginx ES Modules 零依赖 响应式 MIT

简介

基于 Nginx 运行的极简控制台，集成 系统仪表盘、 服务导航、Markdown 博客、 图片画廊 四大模块。所有内容通过 nginx autoindex 或预生成的 index.json 发现，无需任何后端运行时。

本项目衍生自 bastienwirtz/homer， 在长期使用中逐步演变为当前形态。

四大模块

📊
系统仪表盘
8 张实时卡片：设备 / CPU / 内存 / 储存 / 网络 / 电池 / 服务 / 运行时间
🧭
服务导航
分组卡片展示，搜索过滤，安全 URL 白名单跳转
📝
博客阅读器
三栏布局，Markdown 内联渲染 + ToC，HTML 文章新标签页打开
🖼️
图片画廊
网格缩略图，搜索过滤，灯箱放大，加载失败自动隐藏

技术栈

  • 所有 JS 使用 ES Modules（import / export 显式声明依赖）
  • Markdown 解析：marked 引擎 + 白名单 HTML sanitizer + KaTeX 数学公式
  • CSS 策略：CSS 自定义属性 + 3 个响应式断点（1200 / 640 / 400px）
  • Service Worker：分层缓存（network-only / SWR / network-first / cache-first）
  • 仪表盘数据：corn.sh 定时采集系统指标 → dashboard.json → 前端每 10s 轮询
  • 文件发现：gen_index.sh 生成 index.json 优先 → nginx autoindex 降级
  • 安全：文件名 escapeHtml 转义 + URL 白名单验证 + HTML sanitizer 五重过滤

目录结构

# 一键克隆 git clone https://github.com/lost-clouds/Blog-termux.git # 下载依赖库 cd Blog-termux/lib curl -sSLO https://cdn.jsdelivr.net/npm/marked/marked.min.js curl -sSLO https://cdn.jsdelivr.net/npm/katex/dist/katex.min.js curl -sSLO https://cdn.jsdelivr.net/npm/katex/dist/katex.min.css curl -sSLO https://cdn.jsdelivr.net/npm/katex/dist/contrib/auto-render.min.js curl -sSLO https://cdn.jsdelivr.net/npm/github-markdown-css/github-markdown.min.css # 配置 Nginx → 添加 cron → 启动 cp example/Blog.conf $PREFIX/etc/nginx/conf.d/Blog.conf bash gen_index.sh # 生成静态索引 nginx -s reload

部署要素

🖥️
Nginx
Web 服务器，提供 autoindex 目录列表和静态文件服务
⏱️
cron
定时执行 corn.sh 采集系统指标，每 30s 刷新 dashboard.json
📁
Markdown/
放入 .md 文章，博客标签页自动发现并渲染
🖼️
Image/
放入图片，画廊标签页自动展示；thumbnails/ 和 archive/ 会被跳过
